# MyBoq Manifest Module

This is **Module 1** of the MyBoq platform — a fully custom
cloud-native build & runtime system inspired by Bazel, Kubernetes,
and modern microservice platforms.

## Features (MVP 1.5)

✔ YAML manifest parsing  
✔ Strong type-safe Node model  
✔ Version objects  
✔ Dependency parsing  
✔ Directed Acyclic Graph (DAG)  
✔ Cycle detection  
✔ Deterministic topological sort  
✔ Composite node support  
✔ Missing dependency detection
